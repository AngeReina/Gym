Metodos de Pago
Info del Sistema

El sistema soporta pagos mixtos: una sola transaccion puede incluir multiples metodos (Efectivo, Tarjeta y Transferencia simultaneamente).

Al completar el pago, la factura asociada cambia automaticamente a estado PAGADA.

Campos por metodo:
- Efectivo: monto
- Tarjeta: monto, franquicia, tipo
- Transferencia: monto, banco origen, num. transaccion
